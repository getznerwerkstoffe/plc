# TIA Libraries

This folder contains reusable **PLC library templates** for Siemens TIA Portal projects. These libraries are designed to support automation systems at Getzner by providing standardized functionality.

## Included Libraries

### Materialparameter
Handles communication between the PLC and the SCADA system, enabling reliable parameter exchange and source/target value handling.

### Usermanagement
Provides authentication functions and integrates with the Getzner production authentication system for secure user access management.
